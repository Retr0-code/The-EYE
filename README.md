# INSTALLAITION

<b>Linux/Termux</b>
<br>
```sh
git clone https://github.com/Retr0-code/The-EYE
cd The-EYE
pip install -r requirements.txt
python3 TheEYE.py
```

<br>
<b>Windows</b>
<br>
Download archive

Open the cmd in the folder where you unpacked the archive

```sh
pip install -r requirements.txt
.\TheEYE.py
```
